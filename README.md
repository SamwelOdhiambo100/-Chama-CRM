# ChamaCRM
# Chama Manager

A simple Python Flask web application to manage informal savings groups (Chamas). Track member contributions, balances, history, and financial progress.

## Features

* **Add Members:** Easily register new members.
* **Record Contributions:** Log member contributions with amounts and dates.
* **View Balances:** Check individual and all member balances.
* **Persistent Storage:** Data is saved using JSON, ensuring no data loss.
* **Member Progress Tracking:**
    * Visual progress bar to show financial standing.
    * Status indicators (Excellent, Good, Fair, Critical) with color coding.
    * Displays expected contributions based on daily savings goals.
* **Web-based Interface:** Interactive and user-friendly web interface using Flask.
* **Clear Data:** Ability to clear all saved data.

## Getting Started

1.  **Clone the repository:** `git clone https://github.com/SamwelOdhiambo100/ChamaCRM.git`
2.  **Install Flask:** `pip install Flask`
3.  **Run the application:** `python app.py`
4.  **Open in Browser:** Open your web browser and navigate to `http://127.0.0.1:5000/`.

## Usage

Use the web interface to manage your Chama. The site includes forms for adding members, recording contributions, viewing balances, and a table showing member progress.
